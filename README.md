# 📊 حاسبة متوسط سعر السهم

حاسبة بسيطة لحساب متوسط سعر السهم بعد شراءين مختلفين مع إضافة مصاريف البيع والشراء.

## ✨ المميزات
- حساب متوسط السهم لصفقتين
- إضافة مصاريف البيع والشراء
- عرض تفصيلي للتكاليف
- واجهة عربية سهلة الاستخدام

## 🚀 كيفية الاستخدام
1. أدخل سعر وعدد أسهم الشراء الأول
2. أدخل سعر وعدد أسهم الشراء الثاني  
3. أدخل إجمالي مصاريف البيع والشراء
4. اضغط "حساب متوسط السعر"
